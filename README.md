# python-projects

A collection of personal Python projects covering various topics and utilities.

## Contents

- **bank** — Bank account simulation with checking and savings account functionality
- **personal-finance-tracker** — Tools for tracking and managing personal finances
- **inventory-manager** — CLI-based e-commerce inventory manager with CSV persistence (add, view, filter, and delete items)
- **sales-analysis** — Reads sales data from CSV, computes per-row and grand totals, and exports results to JSON, Excel, or CSV
- **python-for-ai** — Learning exercises covering Python fundamentals (lists, tuples, sets, dictionaries, functions, classes, inheritance) and data handling with external APIs
